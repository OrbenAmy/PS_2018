This is the code to reproduce the analyses of Orben and Przybylski (2018), Screens, Teens and Psychological Well-Being: Evidence from three time-use diary studies, Psychological Science.



The code is split into different subscripts:
1. Clean data
2. Get information from data
3a. Run the Specification Curve Analysis
3b. Make the data tables
4. Plot the Specification Curve Analysis
5. Run SCA which is unstandardised
